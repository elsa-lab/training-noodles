1. Installation¶

1.1. Prerequisites¶

  1. Linux-based terminals (For Windows, please see Linux-based Terminal for Windows.)

  2. Python 3.5 or higher

1.2. Install Noodles via PyPI¶

Execute the following command in the terminal:

pip install training-noodles
